Pomoc Pomoc Preuzimanje projekta

Ukoliko nije instaliran, potrebno je instalirati Git na racunaru.
Koraci sa kloniranje projekta sa GitHuba

  • U terminalu Visual Studio Code-a pozicionirati se na folder u koji će biti kloniran projekat
  • Naredbom git clone https://github.com/rva2019/IT51-2016-Petrovic-Milan.git projekat je kloniran sa GitHub-a
 Pokretanje aplikacije

Prije pokretanja frontend apliakcije, pokrenuti backend aplikaciju
 Instalirati Google Chrome plugin: CORS
dodati u Intercepted URLs or URL patterns sledeće URL-ove:

  • http://localhost:8083/
  • http://localhost:4200/

Omogućiti Enable cross-origin resource sharing u Chrome-u. U integrisanom Terminalu VSC-a, pozicionirati se unutar kloniranog projekta i izvršiti sledeće naredbe:

  • npm install - za preuzimanje node-modules (modula potrebnih za pokretanje aplikacije)
  • ng serve -o - pokretanje aplikacije (aplikacija je pokrenuta na portu 4200 i automatski se otvara default browser na linku: http://localhost:4200/)
